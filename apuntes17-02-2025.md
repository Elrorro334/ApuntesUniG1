# En este apunte se mostraran los scrpis para crear indices
#### Creacion de indice
```json 
db.alumnos.createIndex( { nombre:1},{name:"idx_IndiceNombre"})
```
- `1` para orden acendente y menos uno(`-1`) para desendente

#### Consulta de indice 
```json 
db.alumnos.getIndexes()
```
#### Comprobacion de ejecucion de indices 
``` json 
db.alumnos.find({nombre:"maria"}).explain()
```
#### Eliminacion de indices
```json 
db.alumnos.dropIndex("idx_IndiceNombre")
```
#### Indices compuestos 
```json 
db.alumnos.createIndex( { nombre:1,promedio:-1},{name:"idx_IndiceCompuesto"} )
```
